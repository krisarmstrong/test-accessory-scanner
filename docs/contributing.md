# Contributing to iPerf Discovery Utility

## Welcome

Thank you for your interest in contributing!

## Prerequisites

- Python 3.6+
- Network testing environment
- NetAlly Test Accessories (for testing)

## Setup

1. Clone repository
2. No dependencies needed (uses standard library)
3. Run tests: `python -m pytest tests/`

## Development

- Follow PEP 8 guidelines
- Use asyncio for async operations
- Add comprehensive logging
- Test with various network configurations

## Testing

- Test on multiple network ranges
- Verify timeout handling
- Test with actual NetAlly devices
- Check log file output
